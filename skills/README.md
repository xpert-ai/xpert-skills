# Skills Directory

This directory contains example skills grouped by domain.

## Categories
- `creative-design`: Creative and design-oriented skills.
- `development-technical`: Engineering, testing, and automation skills.
- `enterprise-communication`: Skills for brand, messaging, and business communication.
- `document-skills`: Skills for document creation, summarization, and review.

## Skill Layout

Each skill lives in its own folder and must include a `SKILL.md` file.

Example:

```text
skills/
  code-review-checklist/
      SKILL.md
```

## Adding A Skill

1. Copy `template/template-skill/` to a new folder inside the appropriate category.
2. Update the YAML frontmatter in `SKILL.md`.
3. Replace the placeholder instructions with task-specific guidance.
4. Keep the skill self-contained so it can be loaded independently.
